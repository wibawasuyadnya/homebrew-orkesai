# DotAI Homebrew tap

```bash
brew install wibawasuyadnya/dotai/dotai
```

Then type `dotai` (or add `source ~/.config/local-ai/ai-hook.sh` to your
shell rc for the full `ai` command). Docs: https://wibawasuyadnya.github.io/dotai

The formula source of truth lives in the main repo at
[`deploy/homebrew/dotai.rb`](https://github.com/wibawasuyadnya/dotai/blob/main/deploy/homebrew/dotai.rb).
